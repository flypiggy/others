#others

store something like zshrc, zshtheme and so on

###bira+.zsh-theme
my oh-my-zsh theme base on bira.zsh-theme

```
ln -s bira+.zsh-theme ~/.oh-my-zsh/themes
```
set ZSH_THEME="bira+" in .zshrc

###tmux.conf
my tmux conf

```
ln -s tmux.conf ~/.tmux.conf
```

###zshrc
my zshrc file

```
ln -s zshrc ~/.zshrc
```
